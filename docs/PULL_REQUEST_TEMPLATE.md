## Description

Add a description of your PR. Add a **summary** of new changes, and why you made those changes. Please avoid using long paragraphs, and consider bullet points.


## Screenshots

Add any screenshots, where necessary. This is not a strict requirement, but maybe helpful to explain certain things that are otherwise harder with words alone.


## Additional Notes

You may add anything else that you find is important here, otherwise leave blank or N/A.


## Ready Checklist

Make sure you do at least the minimum outlined here, before submitting your PR:

- [ ] Wrote a description of changes
- [ ] Added comments/documentation in your code for legibility
